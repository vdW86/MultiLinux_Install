# MultiLinux_Install
Een Installhulp voor meerdere Distros met Hyprland

De map " Shared" bevat alle scripts die onder alle Distros werken.

########## Installatie OpenSuSe TumbleWeed ##########

Bij OpenSuSe installeer ik de " Server"  versie en (de)selecteer ik de volgende opties:

**** Aanvinken  
	Hyprland (Wayland Compositor)
	Hyperland Extra Components (Plugins and addons)
	32 Bit runtime Environment
	Yast Desktop Utilities
	
****Uitvinken
	Sway Tiling Wayland compositor and related tools
	Yast Server utilities
